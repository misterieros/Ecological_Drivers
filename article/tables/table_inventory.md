# Table inventory for manuscript formatting

Recommended article organization:

| Destination | Content |
|---|---|
| Main Table 1 | Dataset taxonomic breadth, temporal coverage and sampling structure |
| Main Table 2 | Environmental variables and units |
| Main Table 3 | Main model outputs or synthesis of GAM/PCA results |
| Supplementary Tables S1-S... | Extended occurrence counts, complete model summaries, PCA loadings and ancillary remote-sensing tables |

Table exports should be editable, avoid vertical rules, use consistent units, and be cited in the manuscript according to first appearance.
